## 使用原则

1. 只使用公共领域、CC 开放授权或权利人明确允许再利用的图片。
2. 页面内不去除原始署名，不暗示图片作者或来源机构为本项目背书。

## 下载方式

在项目根目录打开 PowerShell，运行：

```powershell
powershell -ExecutionPolicy Bypass -File scripts\download-biography-images.ps1
```

脚本通过 Wikimedia Commons 的 `Special:Redirect/file` 下载最大宽度约 1600 像素的版本，避免把超大原图直接提交到仓库。
